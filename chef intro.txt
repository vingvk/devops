-> bootstrap 
	Once the workstation machine is configured, it can be used to install the chef-client
        on one (or more) nodes across the organization using a knife bootstrap operation. 
	The knife bootstrap command is used to SSH into the target machine, and then do what
	 is needed to allow the chef-client to run on the node.
